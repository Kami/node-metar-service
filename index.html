METAR Weather Information REST Service

REST service which translates METAR weather information into more easy to use machine readable format.

Usage

Sending a single METAR value

curl -X POST -H "Content-Type: application/json" https://api.metarservice.com/metar -d '{"value": "METAR STRING"}'

Sending multiple METAR values

curl -X POST -H "Content-Type: application/json" https://api.metarservice.com/metar -d '{"value": ["METAR STRING 1", "METAR STRING 2", "METAR STRING N"]}'

API Service Limits

Service is free to use, but to prevent abuse, there are some limits in place:

  • Maximum number of requests - 1000 in 24 hours
  • Maximum body size - 2 KB (around ~20 METAR strings)
